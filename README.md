# 赤ちゃんとちょっとお出かけプランナー 👶

赤ちゃんとのお出かけを計画するためのWebアプリケーションです。授乳間隔やおむつ交換のタイミングから、最適な出発時刻を逆算してスケジュールを作成します。

## 🌟 主な機能

- **📝 赤ちゃんの習慣入力**: 授乳間隔、おむつ交換間隔を設定
- **📍 お出かけ計画**: 目的地名、到着時刻、滞在時間、移動時間を入力
- **⏰ 自動スケジュール生成**: 準備開始時刻、授乳・おむつ交換のタイミングを自動計算
- **🧷 おむつ枚数計算**: 必要なおむつの枚数を自動で算出
- **📷 画像保存**: スケジュールを画像として保存可能
- **💬 LINE共有**: 詳細なスケジュールをLINEで簡単に共有

## 🚀 使い方

1. **赤ちゃんの基本情報を入力**
   - 名前（任意）
   - 授乳間隔（分）
   - おむつ交換間隔（分）

2. **お出かけ計画を入力**
   - 目的地の名前（任意）
   - 目的地到着時刻
   - 目的地での滞在時間
   - 移動時間（片道）

3. **スケジュールを計算**
   - ボタンを押すと自動的に最適なスケジュールが生成されます

4. **共有・保存**
   - 📷 画像として保存：スケジュールをPNG画像でダウンロード
   - 💬 LINEで共有：パートナーや家族に詳細を送信
